# estudos-java
# Estudos de Java

Este repositório foi criado para guardar os exercícios que estou fazendo durante o curso de Java do professor Nelio Alves.

Aqui estão exercícios práticos com:

- criação de classes
- objetos
- métodos
- construtores
- encapsulamento (getters e setters)
- enum
- toString()
- manipulação de datas
- ArrayList
- exercícios de Programação Orientada a Objetos (POO)

## Objetivo

O objetivo é acompanhar minha evolução no Java e ir adicionando novos exercícios conforme avanço no curso.

## Organização

Cada pasta representa um exercício ou um pequeno projeto feito durante os estudos.
